# ❓ Problématique

Cette page recense toutes les problématiques que nous avons sur l'infrastructure de **Millenuits**.

Les problématiques ont trois statuts :

⚫ Non traité
🟠 En cours
🟢 Traité

| Titre                              | Statut | Description de la problèmatique                                                                                                                                                                                                                                                                                                                                                                                                      |
| ---------------------------------- | ------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Automatisation installation Debian | ⚫      | L'objectif serait de réussir à automatiser l'installation de Debian sans avoir à configurer le stockage ou les autres éléments à la mains. La piste d'un script est envisagé.                                                                                                                                                                                                                                                        |
| Automatisation post-configuration  | ⚫      | Après chaque installation nous devons remettre en place notre bannière ainsi que d'autres configurations (voir annexe : [Configuration VM Debian](https://ap-bts-sio-louis.github.io/millenuits/ressources/socle-technique/02-linux-debian/Configuration%20type%20machine%20virtuelle%20Debian%2013/)) nous souhaitons gagner du temps dans le déploiement et éviter ses actions répétitives. La piste de Ansible est privilégier.   |
